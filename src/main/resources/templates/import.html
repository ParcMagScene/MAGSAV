Format des fichiers CSV

Le fichier CSV doit contenir les colonnes suivantes :

nom prenom telephone email adresse
Dupont Jean 0123456789 jean.dupont@email.com 123 rue de la Paix, Paris

Le fichier CSV doit contenir les colonnes suivantes :

nom contact telephone email adresse
TechRepair SARL Marie Martin 0987654321 contact@techrepair.fr 456 avenue des Sciences, Lyon

Le fichier CSV doit contenir les colonnes suivantes :

proprietaire produit numero_serie panne detecteur date_entree statut
Jean Dupont iPhone 14 ABC123456 Écran cassé Technicien A 2023-01-01 recu
Notes :
  • Format de date : YYYY-MM-DD
  • Statuts valides : recu, en_cours, termine

Même structure que Dossiers SAV (produit = appareil). Colonnes :

produit numero_serie proprietaire panne statut detecteur date_entree date_sortie
Ordinateur HP 840 G9 SN123456789 Alice Martin Ne démarre plus recu Technicien B 2025-09-01
Produit = champ 'produit' de DossierSAV (aucune table produit séparée). Utilisez ce format si vous avez un export "produits" standard.
